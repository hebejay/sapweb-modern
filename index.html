📜 诗苑 · Poetry

登鹳雀楼

唐 · 王之涣

白日依山尽，

黄河入海流。

欲穷千里目，

更上一层楼。

意指要想看到无穷无尽的美丽景色，应当再登上一层楼。比喻想要取得更大的成功，就要付出更多的努力；要想在某一个问题上有所突破，可以在一个更高的角度审视它；也表达了只有积极向上才能高瞻远瞩。

🛠️ 虚拟机优化技巧 · VMWare Tips

绝招1：禁用 VMEM 文件

编辑对应的 VMWare 虚拟机镜像后缀为 .vmx 的文件，添加一行参数：

mainMem.useNamedFile = "FALSE"

没此参数前虚拟机运行时会在本机产生后缀为 .vmem 的文件。虚拟机上的内存读写 ≈ 本机 VMEM 文件的读写，加上此参数后速度明显提升。但暂停虚拟机速度会变慢，突然断电更难保证数据完整性。

绝招2：全部内存写入 RAM

虚拟机 → Edit(编辑) → Preferences(偏好设定) → Memory(内存)

默认：Allow some virtual machine memory to be swapped

改为：Fit all virtual machine memory into reserved host RAM

⚠️ 此设置需要本机内存足够大，否则适得其反。普通机器用默认参数即可。
